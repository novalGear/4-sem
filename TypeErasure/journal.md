## 1

**Object** обладает properties (доступ по имени).

**Property**: обертка над Value для использования несколькими объектами и добавления метаданных.

**Value**: Использую type erasure через v-table: в конструкторе value известна информация о типе, использую ее чтобы сохранить указатель на нужную таблицу с реализациями функций для данного типа.

Реализация функций для vtable пока находится в vtable_spec, там реализованы затычки для int, string, bool, double
